### 写一篇文章

在 `content/posts/` 下新建 `.md` 文件，顶部写：

```
---
title: "文章标题"
date: 2026-06-07
draft: false
weight: 1
---
```

图片放在 `static/images/`，文章里用 `![](/images/xxx.png)` 引用。
