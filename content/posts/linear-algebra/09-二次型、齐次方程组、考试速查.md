### 二次型

二次型是用矩阵来写二次曲线的工具。

$$
f = \vec{x}^T A \vec{x}
$$

其中 $A$ 是**对称矩阵**（$A^T = A$），对角线是平方项系数，非对角线是交叉项系数的**一半**。

**例：** $f = 2x^2 + 3xy + 4y^2$ 写成矩阵：

$$
f = \begin{bmatrix} x & y \end{bmatrix} \begin{bmatrix} 2 & 1.5 \\ 1.5 & 4 \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix}
$$

#### 配方法化标准型

消去交叉项，得到 $f = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots$

标准型的系数就是矩阵 $A$ 的**特征值**。

#### 正定判定

| 方法 | 条件 |
|:---|:---|
| **特征值法** | 所有特征值 $> 0$ |
| **顺序主子式法** | 所有顺序主子式 $> 0$ |
| **惯性指数法** | 正惯性指数 $= n$ |

**相关概念：**

| 类型 | 条件 |
|:---|:---|
| 正定 | 所有特征值 $> 0$ |
| 半正定 | 特征值 $\geq 0$，且至少有一个为 $0$ |
| 负定 | 所有特征值 $< 0$ |
| 不定 | 特征值有正有负 |

#### 惯性指数

- **正惯性指数 $p$**：标准型中正平方项的个数
- **负惯性指数 $q$**：标准型中负平方项的个数
- **符号差**：$p - q$

**惯性定理：** 一个二次型的正负惯性指数是唯一确定的，与配方法的具体步骤无关。

### 齐次方程组

齐次方程组就是 $AX = 0$（右边全是 0）。

==齐次方程组 $AX = 0$ 有非零解 $\iff$ $\det(A) = 0$（$A$ 不是方阵时，看 $r(A) < n$）==

- $\det(A) \neq 0$ → 只有零解（唯一解 $X = 0$）
- $\det(A) = 0$ → 有无穷多非零解

### 考试速查

#### 试卷名词翻译表

| 试卷上的话 | 翻译 | 要做什么 |
|:---|:---|:---|
| 伴随矩阵 $A^*$ | 求逆的中间工具 | 小 a 换大 A，然后全转置 |
| 增广矩阵 | 系数+常数拼一起 | 右边多一列常数 |
| 用初等行变换求逆 | 把 $A$ 变 $E$，$E$ 变 $A^{-1}$ | $[A \mid E]$ 行变换 |
| 矩阵的秩 | 非零行行数 | 化简到阶梯型，数非零行 |
| 极大无关组 | 最少的向量代表全体 | 行变换看主元列 |
| 用基础解系表示全部解 | 通解 = 特解 + 基础解系 | 找自由变量 |
| 线性相关/无关 | 有没有"多余"的向量 | 算秩或行列式 |
| 能否对角化 | 特征向量够不够 $n$ 个 | 代数重数 = 几何重数？ |
| 是否正定 | 特征值全为正？ | 特征值法或顺序主子式法 |
| 齐次方程组有非零解 | 空间被压扁了 | $\det(A) = 0$ |
| 克莱姆法则 | 用行列式解方程组 | 把右边替换某一列，算行列式比值 |

#### 计算套路

**行列式：** 先看三角矩阵秒杀 → 不能就倍加造 0 → 按 0 最多行/列展开 → 降阶到 2×2

**求逆：** 先算 $\det(A)$ → 初等行变换 $[A|E] \to [E|A^{-1}]$（大题首选）

**解方程组：** 增广矩阵 → 行阶梯型看秩 → 行最简型读解 → 通解 = 特解 + 基础解系

**特征值/特征向量：** $\det(\lambda E - A) = 0$ 求 $\lambda$ → 解 $(\lambda E - A)X = 0$ 求向量

**对角化：** 求特征向量 → 正交化 → 单位化 → 拼成 $P$

#### 常见线性变换

| 变换 | 矩阵形式 | 效果 |
|:---|:---|:---|
| 均匀缩放 $k$ 倍 | $\begin{bmatrix} k & 0 \\ 0 & k \end{bmatrix}$ | 整个空间放大/缩小 |
| 沿 x 轴缩放 $k$ 倍 | $\begin{bmatrix} k & 0 \\ 0 & 1 \end{bmatrix}$ | 水平拉伸 |
| 沿 y 轴缩放 $k$ 倍 | $\begin{bmatrix} 1 & 0 \\ 0 & k \end{bmatrix}$ | 竖直拉伸 |
| 剪切（沿 x 轴） | $\begin{bmatrix} 1 & s \\ 0 & 1 \end{bmatrix}$ | 水平方向歪斜 |
| 旋转 $\theta$ 角度 | $\begin{bmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{bmatrix}$ | 绕原点旋转 |

#### 常见易混淆点

| 容易混淆的 | 正确区分 |
|:---|:---|
| $\|kA\|$ 和 $k\|A\|$ | $\|kA\| = k^n\|A\|$；$k\|A\|$ 只是最后乘一次 |
| $AB$ 和 $BA$ | $AB$ = 先做 $B$ 再做 $A$；一般不相等 |
| 余子式 vs 代数余子式 | 代数余子式 = $(-1)^{i+j} \times$ 余子式 |
| 伴随矩阵 vs 逆矩阵 | 伴随矩阵是求逆的中间工具，不是逆矩阵本身 |
| 相似 vs 合同 | 相似 $B = P^{-1}AP$；合同 $B = P^TAP$ |
| 代数重数 vs 几何重数 | 代数重数 = 特征根的重数；几何重数 = 解空间维度 |
| 行列式 vs 叉积 | 行列式是矩阵属性（缩放）；叉积是两向量运算（带方向面积） |

#### 几何直觉对照表

| 代数概念 | 几何直觉 |
|:---|:---|
| 矩阵 $A$ | 线性变换（空间在动） |
| $\det(A)$ | 面积/体积的缩放倍数 |
| $\det(A) = 0$ | 空间被压扁 |
| 列空间 | 变换后能到达的所有位置 |
| 秩 | 列空间的维度 |
| 零空间 | 被压到原点的那条线 |
| 特征向量 $v$ | 变换中方向不变的向量 |
| 特征值 $\lambda$ | 该方向的缩放倍数 |
| 逆矩阵 | 撤销变换 |
| 单位矩阵 $E$ | 什么都不做的变换 |
| 矩阵乘法 $A \cdot B$ | 先做 $B$，再做 $A$ |
| 矩阵加法 $A + B$ | 两个变换同时作用 |
